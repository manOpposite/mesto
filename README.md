# Mesto
## Что это?

* Учебный проект Яндекс.Практикум
* Сверстан с применением методологии БЭМ
* Применена схема организации файловой структуры Nested

## Возможности:
* Добавлять карточки
* Удалять крточки
* Ставить лайк
* Изменять данные своего профиля

## Стек технологий
* HTML
* CSS
* JAVASCRIPT
